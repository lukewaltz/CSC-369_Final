## Importing Data From Kaggle
1. Sign into Kaggle Account
2. In order to use the Kaggleâ€™s public API, you must first authenticate using an API token. Go to the 'Account' tab of your user profile and select 'Create New Token'. This will trigger the download of kaggle.json, a file containing your API credentials.
3. Place the kaggle.json file in the directory ~/.kaggle/ on Linux/Mac or %USERPROFILE%\.kaggle\ on Windows.
4. In the command line: kaggle datasets download erdemtaha/cancer-data
